# רשימת משימות לפיתוח המנתח הלקסיקלי (Lex / Scanner)

## 1. התעלמות מרווחים והערות (Whitespace & Comments)
- [ ] **רווחים וירידות שורה:** יצירת חוק שמתעלם מרווחים , טאבים (Tabs) וירידות שורה.
- [ ] **ספירת שורות:** הוספת משתנה גלובלי שסופר ירידות שורה (`\n`) כדי שיהיה אפשר להדפיס מספרי שורות בהודעות שגיאה.
- [ ] [cite_start]**הערות בלוק חוקיות:** זיהוי והתעלמות מתוכן שנמצא בתוך בלוק הערות שמתחיל ב-`/%` ומסתיים ב-`%/`[cite: 622].
- [ ] [cite_start]**הערות לא חוקיות:** וידוא שהערות בסגנון C הרגיל (כמו `/* wrong language */`) לא מזוהות כהערות חוקיות וזורקות שגיאה לקסיקלית [cite: 625-626].

## 2. מילות מפתח (Keywords)
[cite_start]יש ליצור Token נפרד עבור כל אחת ממילות המפתח הבאות [cite: 520-536]:
- [ ] `bool`
- [ ] `char`
- [ ] `int`
- [ ] `real`
- [ ] `string`
- [ ] `int*`
- [ ] `char*`
- [ ] `real*`
- [ ] `if`
- [ ] `else`
- [ ] `while`
- [ ] `for`
- [ ] `var`
- [ ] `func`
- [ ] `proc`
- [ ] `return`
- [ ] `null`

## 3. אופרטורים (Operators)
[cite_start]יש לזהות ולהחזיר את האופרטורים הבאים (לחלקם כדאי ליצור Token ייעודי, ולחלקם אפשר פשוט להחזיר את התו עצמו) [cite: 539-548]:
- [ ] אריתמטיים: `+`, `-`, `*`, `/`
- [ ] השוואה: `==`, `>`, `>=`, `<`, `<=`, `!=`
- [ ] לוגיים: `&&`, `||`, `!`
- [ ] [cite_start]פעולות על מצביעים: `^` (Dereference) [cite: 587][cite_start], `&` (Address of)[cite: 589].
- [ ] [cite_start]השמה: `=`[cite: 791].

## 4. תווים מיוחדים וסימני פיסוק (Punctuation)
[cite_start]יש לזהות את התווים הבאים (לרוב פשוט מחזירים את התו עצמו `yytext[0]`)[cite: 618]:
- [ ] נקודה פסיק סוף משפט: `;`
- [ ] נקודתיים להצהרת משתנים: `:`
- [ ] פסיק להפרדת פרמטרים: `,`
- [ ] בר למדידת אורך מחרוזת: `|`
- [ ] סוגריים מסולסלים לבלוקים: `{`, `}`
- [ ] סוגריים עגולים לפונקציות: `(`, `)`
- [ ] סוגריים מרובעים למערכים/מחרוזות: `[`, `]`

## 5. ליטרלים וערכים (Literals)
בניית ביטויים רגולריים (Regex) מדויקים עבור:
- [ ] [cite_start]**בוליאני (Boolean):** הערכים `true` או `false`[cite: 551].
- [ ] [cite_start]**תו בודד (Char):** תו בודד מוקף בגרש יחיד (לדוגמה: `'a'`, `'A'`) [cite: 552-556].
- [ ] **מספר שלם עשרוני (Decimal Int):** רצף ספרות. [cite_start]אסור שיתחיל בספרה `0` אלא אם המספר הוא בדיוק `0`[cite: 564].
- [ ] [cite_start]**מספר שלם הקסדצימלי (Hex Int):** מתחיל ב-`0x` או `0X` ואחריו ספרות או האותיות A-F [cite: 564-565].
- [ ] [cite_start]**מספר ממשי (Real):** תומך במספרים עשרוניים עם או בלי סימן (`+`/`-`), עם או בלי נקודה עשרונית, ותומך בכתיב מדעי (כמו `45.3E-23` או `4.e-67`) [cite: 566-567].
- [ ] **מחרוזת (String):**
  - [cite_start]תחומה במרכאות כפולות (`"`)[cite: 568].
  - [cite_start]אסור שתכלול שבירת שורה (Newline) בתוכה [cite: 569-570].
  - [cite_start]אסור שתכיל את התו מרכאות כפולות בתוכה (אין תמיכה ב-Escape למרכאות כפולות. לדוגמה, `"this is \"invalid\""` זה שגוי) [cite: 570, 573-574].
  - [cite_start]מחרוזת שכוללת `\n` נספרת כשני תווים רגילים (`\` ו-`n`) ולא כתו Escape [cite: 575-580].
- [ ] **מזהים (Identifiers):** שמות משתנים או פונקציות. [cite_start]חייבים להתחיל באות (גדולה או קטנה), ולאחר מכן אפס או יותר אותיות, ספרות או קו תחתון (`_`) [cite: 582-583].
